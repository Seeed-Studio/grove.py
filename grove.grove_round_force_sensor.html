grove.grove_round_force_sensor¶

This is the code for

  • Grove - Grove - Round Force Sensor FSR402

Examples

import time
from grove.grove_round_force_sensor import GroveRoundForceSensor

# connect to alalog pin 2(slot A2)
PIN = 2

sensor = GroveRoundForceSensor(PIN)

while True:
    fsr = sensor.value
    print('FSR Value: {}'.format(fsr), end='')
    if fsr < 10:
        print(" - No pressure")
    elif fsr < 600:
        print(" - Light squeeze")
    else:
        print(" - Big squeeze")
    time.sleep(1.0)
